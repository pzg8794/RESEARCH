## Supervisor and project context

- **Prof. Stefano Cherubin** completed PhD at Politecnico di Milano (2019), dissertation on "Compiler-Assisted Dynamic Precision Tuning." Built TAFFO, an LLVM-based precision tuning framework for approximate computing. Moved to Codeplay as compiler engineer, then Edinburgh Napier as Lecturer, joined NTNU IDI in 2023.
- **TAFFO** (Tuning Assistant for Floating-point and Fixed-point Optimization) is SSG's flagship tool — the key classical precedent SPQR extends to the quantum domain.
- **SPQR** proposes using precision-resource trade-off analysis from classical approximate computing to guide QEC code selection at the compiler IR level.
- **SSG primary venues:** CGO, PLDI, CC (CGO 2026 distinguished paper award).
- **JLM** is SSG's in-house RVSDG-based research compiler — relevant IR perspective.
- **Stim** (Google stabilizer circuit simulator) is the baseline QEC simulation tool.
